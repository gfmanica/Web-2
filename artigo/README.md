## Getting Started

Requisitos:

- Última versão do Node.Js instalado

Execute

- npm install (instalar dependências)
- npm run dev (rodar servidor local)

## Padrões do Projeto

Cores:

- background/white bg-slate-50
- cinza/detalhes/bordas bg-slate-300
- main gb-green-500
- secondary bg-cyan-500
- erro bg-red-400
- warn bg-warn-400
